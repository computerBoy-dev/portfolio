## ‚ú® Key Features

### ü§ñ 1. Self-Aware AI Agent (AARYAN_OS)
* **Powered by Groq & Llama 3:** An embedded terminal chat that knows everything about me.
* **Context-Aware:** It can explain how my projects (Pratyaksha, CodeGap) work in technical detail.
* **Strict Persona:** It refuses to answer general queries (e.g., jokes/weather) and stays strictly professional.

### üì± 2. Fluid Responsiveness
* **Mobile-First Engineering:** Uses `clamp()` based fluid typography. Text scales seamlessly from iPhone SE to 4K Ultrawide monitors.
* **Adaptive Layouts:** The grid system transforms from a single-column stack on mobile to a multi-column command center on desktop.

### üé® 3. Industrial Aesthetics
* **CRT Scanline Effect:** A subtle overlay giving a raw, engineering feel.
* **Glassmorphism Cards:** Sharp-edged, frosted glass UI elements.
* **Live Status Terminal:** A dynamic widget showing location, status, and role.
